

These intermediate files are instructional. 

These files will help the user know how to perpare the ligand and receptor.  

 run.003.dockprep.csh       - script to generate grids
 rec_sph.pdb                - portion of protein residue that is point of covalent attachement.  This is used for calculating spheres.  SG, CB, and CA for Cysine (NZ,CE,and CD for lysine).  
 rec_charged.mol2           - receptor perpared in chimera (hydrogens and charge is added), covalent residue is modeled as nutural covalent ligand is removed.  In this case RAS contains GDP (2-), Ca (2+) ion and coordinating waters. 
 rec_charged_mod.mol2       - receptor modified by deleting sidechain (CB, SG, and asociated hydrogens) of covalent residue (Cys 11).  For lysines this might only delelet (NZ,CE, maybe CD, and asociated hydrogens).  
                              do not re-calculate the charge or add hydrogens. 
 rec.box.pdb                - generated by run.003.dockprep.csh for calculating grids, based on ligand.sph  
 ligand_charged.mol2        - Covalent ligand with part of side chain attached. For cystine it is CB and SG are left attached. Hydrogens are added and partial charges are then calculated using Chimera.   
 ligand_charged_mod.mol2    - hydrogens are removed from CB and SG. 
 ligand_charged_mod2.mol2   - residue name is changed to LIG.  CB becomes D2 and SG becomes D1. 
 ligand.sph                 - generated using pdbtosph (accessory program from dock3.7 distribution)
 ligand.pdb                 - just ligand as pdb, used to cacluated spheres for generating the box
